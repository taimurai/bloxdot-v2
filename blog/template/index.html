Open with the strongest sentence in the piece. The lede paragraph carries more visual weight than body copy, so make it count: state the claim, the question, or the specific moment the post turns on.

Continue with body copy. Paragraphs are set in Fraunces at a comfortable reading size with a 64-character measure. Inline links use the studio's crimson with a thin underline. Body italic is reserved for emphasis or titles, not decoration. Bold is for the rare term you want a scanning reader to stop on.

first section heading.

Headings are lowercase to match the rest of the site, ending in a red dot. Use h2 for major sections and h3 for subsections within them. Both have scroll-margin-top set so the table of contents anchor links land below the fixed nav.

A second paragraph in the same section. Lists work well for breaking up dense passages:

  • List markers are crimson — the same dot device used in titles.
  • Keep items parallel in structure and roughly equal in length.
  • Short lists (three to five items) read better than long ones.

a subsection heading

Subsection headings are slightly heavier and not lowercased — they read more like labels inside a section than like section titles themselves.

A pull quote sits to the left of a thin crimson rule, in Fraunces at display weight. Keep these tight — one or two sentences at most — and use them for moments worth pausing on.

— Attribution, Project name

second section, with a figure.

Figures sit inside the prose column by default. Captions use the monospace eyebrow style so they read as metadata, not as part of the running text.
